# 专注计时网站

一个简洁高效的专注计时工具，支持倒计时和正计时功能，多人联机排行榜，以及社交分享功能。

## 功能特点

- ⏰ **计时功能**：支持倒计时和正计时两种模式
- 📝 **任务管理**：为每个专注时段设置任务名称
- 🏆 **排行榜**：多人联机排行榜，激励持续专注
- 🔗 **社交分享**：支持微信、QQ等社交平台分享
- 📊 **数据统计**：专注时长统计和效率分析
- 📱 **响应式设计**：适配各种设备屏幕
- 🌙 **深色模式**：支持深色模式，保护眼睛

## 使用方法

1. 打开 `index.html` 文件即可开始使用
2. 选择倒计时或正计时模式
3. 设置任务名称（可选）
4. 点击"开始"按钮启动计时器
5. 完成专注后，记录会自动保存

## 数据存储

本应用使用浏览器的 localStorage 存储用户数据，包括：
- 用户信息
- 专注记录
- 设置偏好

数据仅存储在本地，不会上传到服务器。

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- Chart.js (数据可视化)

## 注意事项

- 社交分享功能在实际环境中需要后端支持
- 多人联机排行榜在实际环境中需要WebSocket服务器支持
- 刷新页面会丢失当前正在进行的计时，请确保完成计时后再刷新页面
